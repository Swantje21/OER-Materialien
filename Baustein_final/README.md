# Diese Readme-Datei gilt für Dateien, die zum OER-Lernbaustein gehören.

Zuletzt akutalisiert: 02.03.2023
durch: Jan Torben Neumann

Der Lernbaustein ist im Kontext des Projektseminars "Digital Literacies" im Master and der CAU zu Kiel entstanden.

# Struktur

Es gibt einen Ordner mit Bilder, wie eine Markdown für den Baustein, als auch diese README-Datei.

Abbildungen sind in den Formataen PNG und JPEG anzufinden.

# Dateinamen

Für die Abbildungen gilt die folgende Benennungskonvention:

Abbildungstyp_Inhalt.Dateiendung

Abb = Eine rein inhaltliche Abbildung

Comic = Ein auflockerndes Element
